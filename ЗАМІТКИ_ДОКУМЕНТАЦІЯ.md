# Нова функція: Замітки

## Що додано

Додано нову сторінку **"Замітки"** в додаток TaskSprout з повноцінним функціоналом для швидких записів.

## Розташування

5-та вкладка в нижній навігації (між "Фокус" та "Профіль")

---

## Функції

### 📝 Створення заміток

**Голосовий ввід:**
1. Натисніть кнопку **мікрофон** 🎤
2. Надиктуйте текст замітки
3. Текст з'явиться в полі вводу
4. Натисніть кнопку **зберегти** 💾

**Ввід з клавіатури:**
1. Введіть текст в поле "Нова замітка..."
2. Натисніть кнопку **зберегти** 💾

### 🔍 Пошук заміток

- Поле пошуку розташоване **вгорі сторінки**
- Завжди видиме
- Фільтрує замітки в реальному часі при введенні
- Пошук по всьому тексту замітки
- Кнопка ❌ для очищення пошуку

### 📋 Відображення заміток

**Формат карток (як Google Keep):**
- Картка з легкою тінню
- Закруглені кути (12px)
- Відображаються перші 2 рядки тексту
- Решта тексту обрізається з "..."

**Інформація на картці:**
- Текст замітки (перші 2 рядки)
- Дата створення внизу:
  - "Сьогодні, 14:30"
  - "Вчора, 09:15"
  - "Понеділок, 18:00" (якщо менше тижня)
  - "25.06.2026, 12:00" (якщо більше тижня)

### 👁️ Перегляд повного тексту

**Розгортання (акордеон):**
1. Натисніть на картку замітки
2. Картка розгорнеться на місці
3. Відобразиться весь текст замітки
4. З'явиться кнопка **редагувати** ✏️
5. Натисніть знову щоб згорнути

### ✏️ Редагування заміток

1. Натисніть на замітку (розгорнути)
2. Натисніть кнопку **редагувати** ✏️
3. Відкриється діалогове вікно
4. Відредагуйте текст
5. Натисніть **"Зберегти"** або **"Скасувати"**

### 🗑️ Видалення заміток

**Swipe для видалення:**
1. Проведіть пальцем по замітці **вліво** ← 
2. З'явиться червоний фон з іконкою смітника
3. Замітка буде видалена
4. З'явиться повідомлення "Замітку видалено"

## Технічні деталі

### Нові файли

1. **lib/models/note.dart**
   - Модель даних для замітки
   - Поля: id, content, createdAt, updatedAt

2. **lib/providers/note_provider.dart**
   - Управління станом заміток
   - Методи: додавання, редагування, видалення, пошук

3. **lib/screens/notes_screen.dart**
   - UI екрану заміток
   - Голосовий ввід, картки, акордеон, swipe

### Оновлені файли

1. **lib/database/database_helper.dart**
   - Додано таблицю `notes`
   - Версія БД оновлена: 1 → 2
   - CRUD методи для заміток

2. **lib/screens/main_screen.dart**
   - Додано NotesScreen в список екранів
   - Додано вкладку "Замітки" в навігацію

3. **lib/main.dart**
   - Додано NoteProvider в MultiProvider

### База даних

**Таблиця notes:**
```sql
CREATE TABLE notes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT NOT NULL,
  created_at TEXT NOT NULL,
  updated_at TEXT
)
```

**Міграція:**
- Старі користувачі: автоматичне оновлення БД з версії 1 до 2
- Нові користувачі: таблиця створюється відразу

## Комбінування з іншими функціями

**Замітки + Завдання:**
1. Запишіть ідею в замітки
2. Потім перетворіть в повноцінне завдання вручну
3. Видаліть замітку після створення завдання

**Замітки + Календар:**
- Записуйте дати в замітки
- Створюйте завдання з датами на основі заміток

**Замітки + Профіль:**
- Зберігайте особисту інформацію
- Контакти та телефони
- Коди та паролі (не рекомендується для важливих)
