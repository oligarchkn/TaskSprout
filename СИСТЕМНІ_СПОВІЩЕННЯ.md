# 🔔 Системні сповіщення про прострочені завдання

## Що реалізовано

Додано повноцінну систему **фонових сповіщень** для Android, яка працює навіть коли додаток закритий.

---

## 📱 Як це працює

### Коли додаток **запущений:**
- Перевіряються прострочені завдання
- Оновлюється бейдж біля дзвіночка (🔔 з цифрою)
- Список сповіщень відображається всередині додатка
- **Також** показуються системні сповіщення в шторці Android

### Коли додаток **закритий:**
- WorkManager автоматично перевіряє завдання у фоні
- Створюються системні сповіщення через Android NotificationManager
- Користувач бачить їх у верхній шторці Android
- При натисканні додаток відкривається з фільтром прострочених завдань

---

## ⏰ Розклад перевірки

### Фонова перевірка (коли додаток закритий):
- **7:30** - ранкова перевірка
- **12:00** - обідня перевірка  
- **15:00** - денна перевірка

### Щоденні нагадування:
- **9:00** - нагадування про всі прострочені завдання
- Повторюється щодня поки завдання не виконано

### В реальному часі (коли додаток відкритий):
- Кожні 5 хвилин автоматична перевірка

## 📲 Дія при натисканні

При натисканні на системне сповіщення:
1. ✅ Додаток відкривається
2. ✅ Автоматично переходить на вкладку "Завдання"
3. ✅ Фільтр встановлюється на прострочені завдання
4. ✅ Користувач бачить список всіх прострочених завдань

   - Системні сповіщення для Android/iOS

## 🔐 Дозволи Android

Необхідні дозволи (запитуються автоматично):

- ✅ `POST_NOTIFICATIONS` - показ сповіщень (Android 13+)
- ✅ `RECEIVE_BOOT_COMPLETED` - відновлення після перезавантаження
- ✅ `VIBRATE` - вібрація при сповіщенні
- ✅ `SCHEDULE_EXACT_ALARM` - точний час перевірки
- ✅ `WAKE_LOCK` - пробудження пристрою для перевірки

## 🧪 Тестування

### Як перевірити роботу:

1. **Створіть прострочене завдання:**
   - Додайте завдання з датою в минулому
   - Почекайте кілька секунд
   - Перевірте шторку Android

2. **Перевірте фонову роботу:**
   - Закрийте додаток повністю
   - Створіть тестове завдання з минулою датою через веб
   - Дочекайтеся часу перевірки (7:30, 12:00, 15:00)
   - Або налаштуйте систему на тестовий час

3. **Перевірте групування:**
   - Створіть 6+ завдань з минулою датою
   - Перевірте що показується групове сповіщення

4. **Перевірте натискання:**
   - Натисніть на сповіщення
   - Додаток повинен відкритися на вкладці "Завдання"
   - З фільтром прострочених

## 🐛 Відомі обмеження

1. **Час не точний на 100%**
   - Android може затримати виконання на 5-15 хвилин
   - Залежить від оптимізації батареї

2. **Виробники можуть блокувати**
   - Xiaomi, Huawei, Samsung мають агресивну оптимізацію
   - Потрібно додати додаток в виключення

3. **Не працює на веб/desktop**
   - Системні сповіщення тільки для Android/iOS
   - На веб залишаються лише внутрішні сповіщення

## 📝 Інструкція для користувачів

### Якщо сповіщення не приходять:

**Xiaomi (MIUI):**
1. Налаштування → Додатки → TaskSprout
2. Автозапуск → Увімкнути
3. Економія батареї → Без обмежень
4. Сповіщення → Дозволити

**Huawei (EMUI):**
1. Налаштування → Додатки → TaskSprout  
2. Запуск → Керувати вручну → Все увімкнути
3. Сповіщення → Дозволити

**Samsung (One UI):**
1. Налаштування → Додатки → TaskSprout
2. Батарея → Не оптимізувати
3. Сповіщення → Дозволити

**Стандартний Android:**
1. Зазвичай працює без додаткових налаштувань
2. Перевірте дозвіл на сповіщення в налаштуваннях
