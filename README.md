# TaskSprout

TaskSprout - мобільний застосунок для управління завданнями, створений для студентів та молодих фахівців від BrightDay Labs.

## Особливості

- ✅ **Швидке додавання завдань** - миттєве захоплення ідей та завдань
- 📝 **Редагування завдань** - легке оновлення існуючих записів
- ✔️ **Відмітка про виконання** - простий спосіб відстежувати прогрес
- 🗑️ **Видалення завдань** - швидке прибирання непотрібних елементів
- 🔍 **Фільтрація** - перегляд всіх, активних або виконаних завдань
- 💾 **Локальне зберігання** - ваші дані завжди доступні навіть офлайн
- 🌓 **Темна тема** - підтримка світлої та темної теми
- 📱 **Mobile-first** - оптимізовано для телефонів

## Технології

- **Flutter** - кросплатформенний фреймворк
- **Provider** - state management
- **SharedPreferences** - локальне зберігання даних
- **Material Design 3** - сучасний дизайн

## Запуск проекту

### Вимоги
- Flutter 3.35.3 або новіше
- Dart 3.9.2 або новіше

### Встановлення залежностей
```bash
flutter pub get
```

### Запуск на емуляторі/пристрої
```bash
flutter run
```

### Збірка для різних платформ

**Android:**
```bash
flutter build apk --release
```

**iOS:**
```bash
flutter build ios --release
```

**Web:**
```bash
flutter build web
```

## Основні функції

### Додавання завдання
1. Натисніть кнопку "Add Task" внизу екрану
2. Введіть назву завдання
3. Натисніть "Add Task" для збереження

### Редагування завдання
1. Натисніть на завдання в списку
2. Змініть текст
3. Натисніть "Save"

### Відмітка як виконане
- Натисніть на checkbox ліворуч від завдання

### Фільтрація завдань
Використовуйте фільтри вгорі екрану:
- **All** - всі завдання
- **Active** - лише активні
- **Completed** - лише виконані

### Видалення завдань
- Натисніть іконку кошика праворуч від завдання
- Підтвердіть видалення

### Зміна теми
- Натисніть іконку сонця/місяця в AppBar для перемикання між світлою та темною темою
