# 📱 Встановлення TaskSprout на Android

## 📦 Готовий APK файл

**Файл:** `app-release.apk`  
**Розташування:** `build\app\outputs\flutter-apk\app-release.apk`  
**Розмір:** 50.5 MB  
**Версія:** 1.0.0+1

---

## 🚀 Інструкція з встановлення

### Крок 1: Перенесення APK на телефон

#### Спосіб 1: Через USB кабель
1. Підключіть Android телефон до комп'ютера через USB
2. На телефоні оберіть режим **"Передача файлів"** (File Transfer)
3. Відкрийте телефон як диск на комп'ютері
4. Скопіюйте `app-release.apk` в папку **Download** на телефоні

#### Спосіб 2: Через хмарне сховище
1. Завантажте APK на Google Drive / Dropbox / OneDrive
2. Відкрийте посилання на телефоні
3. Завантажте файл

#### Спосіб 3: Через месенджер
1. Надішліть `app-release.apk` собі в Telegram / Viber
2. Завантажте файл на телефоні

#### Спосіб 4: Через email
1. Надішліть APK собі на пошту
2. Відкрийте лист на телефоні
3. Завантажте вкладення

---

### Крок 2: Увімкнення встановлення з невідомих джерел

При першій спробі встановлення Android покаже повідомлення:

**"З міркувань безпеки ваш телефон не може встановлювати невідомі додатки з цього джерела"**

**Що робити:**
1. Натисніть **"Налаштування"** (Settings)
2. Знайдіть пункт **"Встановлення невідомих додатків"** або **"Встановити невідомі додатки"**
3. Оберіть додаток через який встановлюєте (наприклад: File Manager, Chrome, Gmail)
4. Увімкніть перемикач **"Дозволити з цього джерела"**
5. Поверніться назад кнопкою "Назад"

**Залежно від версії Android:**
- **Android 8+:** Дозвіл для кожного додатку окремо
- **Android 7 і нижче:** Загальний дозвіл в Налаштування → Безпека → Невідомі джерела

---

### Крок 3: Встановлення APK

1. Відкрийте папку **Downloads** через **File Manager** (Мої файли)
2. Знайдіть файл **app-release.apk**
3. Натисніть на файл
4. Натисніть **"Встановити"** (Install)
5. Зачекайте завершення встановлення (5-10 секунд)
6. Натисніть **"Відкрити"** (Open) або **"Готово"** (Done)

---

### Крок 4: Перший запуск

При першому запуску TaskSprout може запросити дозволи:

#### 🎤 Дозвіл на мікрофон
**Для чого:** Голосове додавання завдань  
**Обов'язково:** Ні (можна пропустити)

#### 📁 Дозвіл на сховище
**Для чого:** Збереження бази даних завдань  
**Обов'язково:** Так (інакше дані не збережуться)

**Рекомендація:** Натисніть **"Дозволити"** (Allow) для всіх запитів.

## 🔧 Видалення додатку

Якщо потрібно видалити:

1. Налаштування → Додатки → TaskSprout
2. Натисніть **"Видалити"**

**Увага:** При видаленні всі завдання будуть втрачені!

---

## 🆘 Вирішення проблем

### Проблема: "Не вдається встановити додаток"

**Рішення 1:** Увімкніть дозвіл на встановлення (див. Крок 2)

**Рішення 2:** Перевірте вільне місце на телефоні (потрібно ~100 MB)

**Рішення 3:** Видаліть стару версію TaskSprout, якщо вона є

### Проблема: "Додаток не відкривається"

**Рішення 1:** Перезавантажте телефон

**Рішення 2:** Видаліть і встановіть знову

**Рішення 3:** Перевірте версію Android (потрібно Android 5.0+)

### Проблема: "Завдання не зберігаються"

**Рішення 1:** Надайте дозвіл на доступ до сховища

**Рішення 2:** Перевірте, чи не ввімкнено режим очищення даних при закритті

**Рішення 3:** Перевстановіть додаток

### Проблема: "Не працює голосовий ввід"

**Рішення:** Надайте дозвіл на використання мікрофону:
- Налаштування → Додатки → TaskSprout → Дозволи → Мікрофон
